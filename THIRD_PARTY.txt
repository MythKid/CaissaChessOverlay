1. Stockfish  (bundled binary: engine/stockfish.exe)
-----------------------------------------------------
Version bundled : Stockfish 17.1 (Windows x86-64-avx2 build)
Obtained from   : https://github.com/official-stockfish/Stockfish/releases/tag/sf_17.1
Project site    : https://stockfishchess.org
License         : GNU General Public License v3 (or, at your option, any
                  later version) — see https://www.gnu.org/licenses/gpl-3.0.txt

Stockfish is bundled here UNMODIFIED. Its complete corresponding source code
for the exact version above is publicly available at the release tag URL
given. Caissa Chess Overlay communicates with it as a separate operating-
system process over the standard UCI (Universal Chess Interface) protocol —
it does not link against Stockfish's code.
